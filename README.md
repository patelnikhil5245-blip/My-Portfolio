# My Portfolio

A React/Vite portfolio website for Nikhil Kumar.

## Overview

This repo contains a personal portfolio built with React and Vite. The site is configured for GitHub Pages deployment under `https://patelnikhil5245-blip.github.io/My-Portfolio/`.

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

The app uses the `base` path defined in `vite.config.js` for GitHub Pages. When deployed, the site should be available at:

`https://patelnikhil5245-blip.github.io/My-Portfolio/`

## Project Structure

- `src/` - React application source files
- `public/` - static assets and JSON data
- `vite.config.js` - Vite configuration
